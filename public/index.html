• Animes

    • /animes
      • Tipo: GET
      • Params: limit
      • Auth: false
      • Desc: retorna uma lista com todos os animes em ordem aleatória
    • /animes
      • Tipo: POST
      • Params:
      • Auth: true
      • Desc: cria um anime, exemplo
    • /animes/:name
      • Tipo: GET
      • Params: name
      • Auth: false
      • Desc: retorna todos os dados de um anime
    • /animes/:name
      • Tipo: DELETE
      • Params: name
      • Auth: true
      • Desc: remove um anime
    • /animes/:name
      • Tipo: PATCH
      • Params: name
      • Auth: true
      • Desc: atualiza um anime
    • /animes/sort/:order
      • Tipo: GET
      • Params: order, limit
      • Auth: false
      • Desc: retorna todos os animes ordenados pelo parâmetro especificado
    • /animes/card/random
      • Tipo: GET
      • Params: limit
      • Auth: false
      • Desc: retorna uma lista com todos os animes em ordem aleatória contendo as infos de card apenas
    • /animes/card/:name
      • Tipo: GET
      • Params: name
      • Auth: false
      • Desc: retorna os dados de um anime na forma de card
    • /animes/card/sort/:order
      • Tipo: GET
      • Params: order
      • Auth: false
      • Desc: retorna todos os animes ordenados pelo parâmetro especificado contendo as infos de card apenas
    • /animes/image/:name
      • Tipo: PATCH
      • Params: name
      • Auth: true
      • Desc: atualiza as imagens de um anime

  • folder: path or query param

  • nickname: path param
